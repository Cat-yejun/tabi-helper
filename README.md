# 旅 Tabi · 일본 여행 도우미

영수증 가계부, 자연어 일정표, 길찾기/대중교통 안내, 사진·실시간 번역, 여행 비서까지
일본 여행 중 필요한 기능을 한 곳에 모은 모바일 우선 웹앱입니다.
이메일 링크로 간편하게 로그인하면 내 데이터가 안전하게 저장됩니다.

## 로그인
이메일 주소만 입력하면 비밀번호 없이 로그인 링크(매직링크)를 받아 로그인합니다.
모든 일정·가계부·번역 기록은 로그인한 사용자별로 따로 저장돼요.

## 주요 기능

### 🏠 홈
이번 여행 총 지출과 다가오는 일정을 한눈에 보고, 자주 쓰는 기능으로 바로 이동합니다.

### ✦ 일정
가고 싶은 곳과 분위기를 자연어로 설명하면 동선을 고려해 일정표를 만들어줍니다.
> 예) "첫날은 오타루 운하랑 오르골당, 둘째날은 삿포로 시내 맥주박물관이랑 라멘요코초"

- 날짜별 타임라인으로 표시되며 장소·시간·메모를 자유롭게 추가/수정/삭제
- **일정 사이마다 "다음 장소까지 이동" 요약**: 어떤 전철·버스를 타는지, 소요시간·요금을 바로 확인하고, 탭하면 상세 경로로 연결
- **각 장소에서 "현재 위치에서" 길찾기**: 지금 내 위치를 출발지로 바로 경로 안내

### 📷 가계부
영수증을 **촬영하거나 앨범에서 골라** 올리면 가게·품목·금액·날짜·분류가 자동 정리됩니다.
인식 결과는 저장 전 모두 수정 가능하고, 직접 입력으로도 기록할 수 있습니다.

### ➤ 길찾기
출발지·도착지를 입력하면 지도에 경로가 표시되고, 대중교통은 **어떤 노선을 어느 역에서
타서 몇 정거장 가는지** 단계별로 안내합니다. 도보·자동차 모드도 지원합니다.

### あ 번역
두 가지 방식으로 사용합니다.
- **사진**: 메뉴판·표지판을 촬영하거나 앨범에서 골라 올리면 말풍선으로 원문·발음·번역·설명 표시
- **실시간**: 카메라로 비추기만 해도 번역. 화면을 탭할 때만 번역하거나(탭 모드), 자동으로 주기적으로 번역(자동 모드) 중 선택

### 💬 비서
교통·맛집·일본어 표현·환율 등 여행 중 궁금한 점을 자유롭게 물어보는 채팅 비서입니다.

## 기술 스택
Next.js (App Router) · Supabase (DB·인증·스토리지) · Anthropic API · Google Maps API · Vercel
