# 旅 Tabi · 일본 여행 도우미

영수증 가계부, 자연어 일정표, 길찾기/대중교통 안내, 사진 번역, 여행 비서까지
일본 여행 중 필요한 기능을 한 곳에 모은 모바일 우선 웹앱입니다.

## 주요 기능

### 🏠 홈
이번 여행 총 지출과 다가오는 일정을 한눈에 보고, 자주 쓰는 기능으로 바로 이동합니다.

### ✦ 일정
가고 싶은 곳과 분위기를 자연어로 설명하면 동선까지 고려해 일정표를 만들어줍니다.
> 예) "첫날은 오타루 운하랑 오르골당, 둘째날은 삿포로 시내 맥주박물관이랑 라멘요코초"

생성된 일정은 날짜별 타임라인으로 표시되며, 장소·시간·메모를 자유롭게 추가/수정/삭제할 수 있습니다.

### 📷 가계부
영수증을 촬영하면 가게 이름, 구매 품목, 금액, 날짜, 카테고리가 자동으로 정리됩니다.
인식 결과는 저장 전 모두 수정 가능하며, 직접 입력으로 새 기록을 추가할 수도 있습니다.

### ➤ 길찾기
출발지와 도착지를 입력하면 지도에 경로가 표시되고, 대중교통의 경우
**어떤 노선을 어느 역에서 타서 몇 정거장 가는지** 단계별로 안내합니다.
도보·자동차 모드도 지원합니다.

### あ 번역
표지판이나 메뉴판을 촬영하면 사진 위 말풍선으로 원문·발음·번역·간단한 설명이 표시됩니다.
번역 기록은 자동으로 저장되어 다시 찾아볼 수 있습니다.

### 💬 비서
교통, 맛집, 일본어 표현, 환율, 여행 중 생기는 궁금한 점을 자유롭게 물어보는 채팅 비서입니다.

## 기술 스택
Next.js (App Router) · Supabase · Anthropic API · Google Maps API · Vercel
